## Lookup modes

Get company logos using the identifier you have:

- **[By domain](/logo-images/get)** - Most reliable when you have verified domains (`shopify.com`)
- **[By stock ticker](/logo-images/ticker)** - Perfect for financial apps (`AAPL`, `TSLA`)
- **[By crypto symbol](/logo-images/crypto)** - Cryptocurrency logos (`BTC`, `ETH`)
- **[By ISIN](/logo-images/isin)** - Global security identification (`US0378331005`)
- **[By company name](/logo-images/name)** - When you only have brand names (`Shopify`, `Tesla`)

### PNG instead of JPG

```html
<img
  src="https://img.logo.dev/shopify.com?format=png"
  alt="Shopify logo in PNG format"
/>
```

In some cases, PNGs with limited color palettes will result in smaller image files with less compression. PNGs support transparency, although not all logo images will be compatible.

### Light and dark modes

```html
<img
  src="https://img.logo.dev/apple.com?theme=dark&format=png"
  alt="Apple logo for dark backgrounds"
/>
```

The `theme` parameter adjusts logo colors for visibility on your background.

- `theme=dark`: Use when displaying logos on dark backgrounds. Inverts light-colored logos (e.g., white logos become black).
- `theme=light`: Use when displaying logos on light backgrounds. Inverts dark-colored logos (e.g., black logos become white).

This feature requires images with transparency.

### Resizing images

```html
<img
  src="https://img.logo.dev/shopify.com?size=80"
  alt="Shopify logo resized to 80px"
/>
```

Our image CDN will resize to any valid integer. We recommend staying under 600px when using raster images like `jpg` and `png`. There is a max size of 800px. Any sizes above that will be clipped to 800px.

### Retina images

```html
<img
  src="https://img.logo.dev/shopify.com?token=LOGO_DEV_PUBLISHABLE_KEY&size=80&retina=true"
  alt="Shopify logo optimized for retina displays"
  width="80"
  height="80"
/>
```

The `retina=true` parameter doubles the source image resolution (160px) while displaying at the specified size (80px). This creates sharp images on high-density displays. Without this parameter, logos may appear blurry.

### Fallback images

In the case where no image is found for a given domain, the API will return a black and white monogram of the first letter of the domain with HTTP Status Code `200 OK`.

<Frame caption="https://img.logo.dev/example.com">
  <img
    alt="Example fallback monogram"
    width="128"
    height="128"
    src="https://img.logo.dev/example.com?token=live_6a1a28fd-6420-4492-aeb0-b297461d9de2"
  />
</Frame>

If you want to suppress these monograms, use the parameter `fallback=404` to receive an empty response and HTTP Status Code `404 Not Found`.
