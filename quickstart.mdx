## Instant logo delivery

Logo.dev is the company logo API that just works. One endpoint, 18M companies, instant delivery.

```html
<img src="https://img.logo.dev/apple.com" alt="Apple logo" />
```

That's it. No API key needed for basic usage.

## Format control

Get logos in the format that works best for your use case:

```html
<!-- Modern format with transparency -->
<img src="https://img.logo.dev/github.com?format=webp" />

<!-- Maximum compatibility -->
<img src="https://img.logo.dev/github.com?format=png" />

<!-- Vector when available -->
<img src="https://img.logo.dev/github.com?format=svg" />
```
